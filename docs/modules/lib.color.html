Color.new (r, g, b)
Creates a new color from RGB space Please note that this uses values in the range of 0 to 255.

Parameters:

  • r number The red component of the color
  • g number The green component of the color
  • b number The red component of the color

Returns:

    Color The new color

Color:to01 (returnTable)
Converts the color to a LÖVE compatible color, in the 0 to 1 range

Parameters:

  • returnTable boolean If the color should return a table, default true

Returns:

  1. number The red component
  2. number The green component
  3. number The blue component
  4. number The alpha component

Or

    {number, ...} A table of the components
